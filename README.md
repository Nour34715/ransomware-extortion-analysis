# Ransomware Extortion Analysis – LotusCare Case Study

## Overview

This project presents a professional cybercrime analysis of ransomware extortion strategies using the fictional LotusCare Services incident as the primary case study.

The presentation examines how modern ransomware groups use data theft, leak threats, operational disruption, and psychological pressure tactics to extort organisations and victims. The project also compares the LotusCare incident with multiple real-world ransomware attacks to critically evaluate the broader implications of ransomware operations.

## Project Purpose

The purpose of this project is to analyse the evolving nature of ransomware extortion and evaluate its implications for organisations, investigators, regulators, and affected individuals.

The project was developed as part of advanced studies in cybercrime and cybersecurity analysis.
